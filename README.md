# Auktas Guides

This repo stores reusable static guide pages.

## Structure

- `index.html` — guide catalog / launcher
- `<guide-name>/index.html` — one self-contained guide page per folder

## Current guides

- `best-online-sportsbooks-usa-2026/`

## Naming rule

Create each new guide in its own slug folder, for example:

- `best-online-sportsbooks-usa-2026/`
- `best-casino-bonuses-usa-2026/`
- `mobile-betting-apps-comparison-2026/`

All guide pages should keep outbound CTA buttons disabled unless explicitly requested.
